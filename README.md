# FITTY

Fitty is a lightweight fitness performance platform to log daily workouts, track progress metrics, and
receive trainer-style exercise recommendations.

## Run locally

You can serve the static site with any local web server. Pick one option below.

### Option A: Python (if installed)

```bash
python -m http.server 8000
```

### Option B: Node.js (if installed)

```bash
npx serve . -l 8000
```

### Option C: No web server (quick preview)

Open `index.html` directly in your browser by double-clicking it.

> Note: If Windows shows **"Python was not found"**, install Python from
> https://www.python.org/downloads/ or use Option B.

Then open `http://localhost:8000` in your browser (Options A/B).
Then open `http://localhost:8000` in your browser.
